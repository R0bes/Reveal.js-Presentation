How to make this presentation

Setup a reveal.js project

There are 3 options presented at revealjs.com/installation

  • Basic (recommended)
  • Full (for developement)
  • As node dependency (limited use)

Basic Setup for a reveal.js project

  • Download the latest reveal.js version
  • Unzip and replace the example contents in index.html with your own
  • Open index.html in a browser to view it
