How to make this presentation

Setup a reveal.js project

There are 3 options at the homepage.

We use the basic one.

  • Basic (recommended)
  • Full (for developement)
  • As node dependency (limited use)

Basic Setup

  • Download the latest reveal.js version
  • Extract all files to your dest directory
  • run npm install && npm start in dest directory
  • Open localhost:8000 in your browser

html document

  • default is index.html
  • other pages (localhost:8000/demo.html)
  • html document
    • head
      • metadata (encoding, viewport)
      • title
      • styling (css files)
    • body
      • content
      • scripts (reveal, plugins, initialization)
